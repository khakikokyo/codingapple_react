# Project 생성

[Create React App](https://create-react-app.dev/) 라이브러리 -

1. 작업용 폴더 생성
2. 터미널

```bash
$ npx create-react-add 프로젝트명
```

3. vscode 프로젝트 폴더 오픈

# Project file 설명

- `node_modules`: 라이브러리 코드 보관함
- `public`: static 파일 보관함
- `src`: 코드 짜는 곳
- `package.json`: 프로젝트 정보
